# coffee-shop

## Project setup

```
npm install
```

Install JSON Server

```
npm install -g json-server@0.17.4    # NPM
yarn global add json-server@0.17.4   # Yarn
pnpm add -g json-server@0.17.4       # PNPM
```

Start JSON Server

```bash
json-server --watch db.json
```
